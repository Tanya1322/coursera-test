WEB DEVELOPMENT

HTML
HTML provides the structure, i.e. the web components that the html document have, like 1 heading, 2 paragraphs and a footer. It does not tell us about anything that how these documents are visually laid out, how they look like, what color they are, what font size they are. It only tells us what the components are.
CSS
The colors and style is the role of CSS. So colors, layouts,font sizesthat is any stylistic type of things. For eg: If we put a heading in our HTML document, it will tell us what colour, what font size it is, etc.
Javascript
The job of javascript is to provide behaviour and provide functionality. It adds functionality to our page. For eg: what happens when an HTML document finishes loading on the browser, or what happens when we click on one of its heading, etc.
